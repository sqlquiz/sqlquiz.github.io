Visit the live application at: [https://sqlquiz.github.io](https://sqlquiz.github.io)

## Features

- Interactive SQL quizzes to test and improve your knowledge
- Instant feedback on answers
- Clean and intuitive user interface
- No server-side setup required — runs entirely in the browser
- Covers a variety of SQL topics and difficulty levels

## Getting Started

### Prerequisites

- A modern web browser
- [Git](https://git-scm.com/) (for cloning the repository)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sqlquiz/sqlquiz.github.io.git
   ```

2. Navigate to the project directory:

   ```bash
   cd sqlquiz.github.io
   ```

3. Open `index.html` in your preferred web browser:

   ```bash
   open index.html
   ```

   Or simply double-click the `index.html` file.

## Usage

1. Open the application in your browser.
2. Select a quiz or topic you want to practice.
3. Read each question carefully and select or type your answer.
4. Submit your answer to receive instant feedback.
5. Review your results and learn from any mistakes.
